This is a simplified one-page web app template. Simply copy RUNME.sh script filename, paste it to the Terminal app and hit Enter (you should install when-changed pip package first and npm package http-server would be highly recommended):

./RUNME.sh

The page and all related css/js will be rebuilt and everything will be automatically reloaded after each change in one of the files in ‘assets’ folder. By default, it uses npm's http-server which runs at 8080 port (the browser window should pop-up as you run it). You can use any other server, just remember to point the web root to its web sub-directory. by using app.log feature you can activate a little debug console. It both works fine to debug on a mobile device, just open the same web address as written
